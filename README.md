# 랜덤 플래닛 (Random Planet)

> 캐릭터 자체가 가챠다 — 매번 랜덤으로 굴려진 캐릭터로 도전하는 로그라이트 액션

메이플스토리 월드 글로벌 개발 콘테스트 제출작 · **리롤 스튜디오 (Re:Roll Studio)**

---

## 개요

| | |
|---|---|
| **플랫폼** | 메이플스토리 월드 (mLua + Unity 기반) / PC·모바일 크로스플랫폼 |
| **장르** | 로그라이트 액션 / 핵 앤 슬래시 |
| **플레이 모드** | 싱글 플레이(솔로) 전용 |
| **타깃 플레이타임** | 1런 약 1시간 |
| **사용 리소스** | 메이플스토리 5차 전직 이전 에셋 전체 |

캐릭터의 스탯·특성·스킬이 전부 랜덤으로 결정되는 로그라이트. 같은 판이 두 번 없고, 잘 뽑은 캐릭터로 더 강한 보스를 순차 클리어해나가는 것이 목표다. 메이플 IP 친화 요소(옛날 직업 스킬·전직·이능·마노·달팽이)를 살리면서, 랜덤/가챠 시스템 자체를 게임의 핵심 메커니즘으로 삼았다.

## 핵심 시스템

캐릭터는 3개의 독립된 100종 풀로 구성된다.

| 레이어 | 개수 | 역할 | 획득 시점 |
|---|---|---|---|
| **특성 (Trait)** | 100종 (5풀 × 20) | 캐릭터의 정체성 — 상시 배경 보정 | 생성 시 랜덤 (+3차 전직 시 1회 추가) |
| **패시브 스킬** | 100종 | 본인 전투 기믹 — 발동·트리거형 | 전직 시 3택 1 (4차까지 최대 4개) |
| **링크스킬** | 100종 | 링크 캐릭터가 아군에게 주는 지원 | 생성 시 랜덤 배정, 15레벨 이상 링크 시 활성화 |

- **캐릭터 생성**: 하루 최대 6개, 스탯·특성 전부 랜덤
- **공격대**: 플레이어블 1 + 링크 2 (총 3명)
- **전직**: 총 4차 (1·5·13·18레벨), 매 전직마다 메이플 옛날 직업 스킬 제시
- **이능**: 3레벨 단위(3·6·9·12·15·18·21) 총 7회, 스탯 재분배 / 추가 스탯 / 특성 변경 중 선택
- **사망**: 영구 삭제(로그라이트 확정), 계정 영구 재화는 유지되어 메타 강화로 누적

## 특성 풀 (가챠 레어도 구조)

| 풀 | 출현 확률 | 성격 |
|---|---|---|
| Hidden — good | 15% | 빌드를 캐리하는 강특성 |
| Hidden — bad | 5% | 강한 제약 + 강한 대가의 고위험 고보상 |
| average | 50% | 무난한 안전 보정 |
| bad | 18% | 빌드에 부담을 주는 핸디캡 |
| Gamble | 12% | 잘 터지면 OP, 안 터지면 핸디캡 |

풀 내부는 균등 추첨(각 20종 → 1/20), 소프트/하드 천장으로 무한 하락을 방지한다.

## 진행 상황 / 미결 항목

현재 기획 단계이며, 아래 항목은 플레이테스트 이후 확정 예정이다.

- [ ] 경험치 테이블·난이도 배율 — 신규 타깃 플레이타임(1시간)에 맞춘 재산정
- [ ] 특성·패시브·링크스킬 300종 수치화 (계수·확률·쿨다운)
- [ ] 일일 생성 한도(6)와 보관함 슬롯(6) 충돌 — 슬롯 관리 흐름 설계
- [ ] 16~21레벨 해금 조건 구체값
- [ ] 보스 구성 큐레이션
- [ ] 큐브 시스템 도입 여부 (메이플 월드 개발 환경 확인 후 결정)
