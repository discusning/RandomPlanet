# 게임 기획서 (콘테스트 제출본)

**플랫폼**: 메이플스토리 월드 (글로벌 개발 콘테스트)
**개발 환경**: mLua 스크립팅 + Unity 기반 / PC·모바일 크로스플랫폼
**사용 가능 리소스**: 메이플스토리 5차 전직 이전 에셋 전체 (2024년 7월 약관 개정)

---

## 0. 게임 개요

| 항목 | 내용 |
|------|------|
| 가제 | 랜덤 플래닛 (RANDOM PLANET) |
| 장르 | 로그라이트 액션 / 핵 앤 슬래시 |
| 한 줄 소개 | 캐릭터 자체가 가챠다 — 매번 랜덤으로 굴려진 캐릭터로 도전하는 로그라이트 |
| 핵심 재미 | 스탯·특성·스킬이 전부 랜덤 → 같은 판이 없음. 강캐를 뽑고 빌드를 완성해가는 재미 |
| 타깃 플레이타임 | 1런 약 1시간 |
| 플레이 모드 | 싱글 플레이(솔로) 전용 |
| 메타 목표 | 더 강한 보스를 순차 클리어 / 계정 영구 강화 누적 |

**콘테스트 어필 포인트**: 메이플 IP 친화 요소(옛날 직업 스킬·전직·이능·마노·달팽이)를 그대로 살리면서, 랜덤 시스템을 게임의 중심 메커니즘으로 끌어올린 로그라이트. 메이플스토리 5차 이전 에셋·보스를 적극 활용해 IP 친숙도를 극대화.

## 1. 핵심 게임 루프

### 1-1. 메타 루프 (로비)

1. 캐릭터 생성 — 하루 최대 6개, 스탯·특성 전부 랜덤
2. 보관 — 캐릭터 선택창(최대 6칸)에서 관리
3. 공격대 편성 — 플레이어블 1 + 링크 2
4. 입장 — 토벌 전장 / 몬스터 게이트 선택
5. 정산 — 클리어 보상으로 영구 재화 획득 → 계정 강화 → 1번으로 회귀

> 캐릭터가 전부 소진되고 당일 생성권도 없으면 다음 날 생성권 회복까지 대기.

### 1-2. 인게임 루프 (전장)

```
스폰
 → 좌우 포털에서 몬스터 웨이브 등장
 → 처치 시 스탯 구슬 / 경험치 드랍
 → 레벨업
 → [5·13·18레벨] 2·3·4차 전직 + 전직 퀘스트 보스 클리어
 → [3레벨 단위] 이능 선택
 → 중간 보스 난입 (막대한 경험치)
 → 사냥터 클리어  또는  캐릭터 사망 → ENDING
```

## 2. 캐릭터 시스템

### 2-1. 캐릭터 생성 및 공격대

- 캐릭터 선택창 최대 6칸 — 모든 슬롯이 플레이어블 속성을 가짐
- 그 중 1개를 선택해 직접 조작 (플레이어블 캐릭터)
- 나머지 중 최대 2개를 링크로 지정 → 공격대 총 3명 구성
- 링크 캐릭터는 15레벨 이상일 때 배정된 링크스킬을 공격대에 제공 (2-8 참조)
- 공격대에 플레이어블 1개만 있어도 도전 가능
- 캐릭터 생성은 하루 최대 6개 (매일 초기화)
- 캐릭터 사망 → 해당 캐릭터 1개만 영구 삭제 (링크 캐릭터는 생존)
- 모든 캐릭터 소진 + 당일 생성권 없음 → 다음 날 대기

> 일일 생성 한도(6개)가 보관함 슬롯 수(6칸)와 동일해짐 — 하루치를 전부 생성하면 보관함이 가득 찬다. 노후 캐릭터 정리(링크 전환·방출) 동선을 함께 설계할 필요 있음 (부록 참조).

### 2-2. 랜덤 스탯

생성 시 힘 / 민첩 / 지능 / 행운 4종에 전부 랜덤 분배.

| 스탯 | 주 효과 | 부 효과 |
|------|---------|---------|
| 힘 (STR) | 물리 공격력 ↑ | 넉백 저항 ↑ |
| 민첩 (DEX) | 공격 속도 ↑, 명중 ↑ | 이동 속도 ↑ |
| 지능 (INT) | 스킬 데미지 ↑, 최대 MP ↑ | 스킬 쿨다운 감소 |
| 행운 (LUK) | 크리티컬 확률 ↑, 크리티컬 데미지 ↑ | 회피 소폭 ↑ |

> 행운(LUK)은 순수 전투 스탯. 드랍률 등 파밍 수치에는 관여하지 않음.

### 2-3. 특성 (Trait)

생성 시 Hidden(good/bad) / average / bad / Gamble 풀에서 랜덤 지정.
3차 전직(13레벨) 시 추가로 1회 더 추첨.

> **풀 선택 확률은 8-2 참조** (average 50% / bad 18% / Hidden-good 15% / Gamble 12% / Hidden-bad 5%).
> 풀이 정해지면 그 안에서 **균등 추첨**하므로, 각 풀 20종이면 풀 내 단일 특성 출현 확률은 1/20(5%)로 고르게 깔림.

#### Hidden — good (출현 확률 15%)

build-defining 강특성 풀. 잘 뽑으면 빌드의 핵심이 됨.

| 특성 | 효과 |
|------|------|
| 상인 | 아이템 드랍률 2배 |
| 버서크 | 초당 -3% 체력 / 체력 낮을수록 공격력 ↑ / 타격 시 흡혈 / 회복 아이템 적용 불가 |
| 링크 마스터 | 링크 패시브 효율 +20% |
| 시간여행자 | 경험치 추가 획득 |
| 부활자 | 사망 시 1회 부활 |
| 먹보 | 회복 아이템 회복량 증가 |
| 분신(도플갱어) | 일정 시간마다 분신 소환, 분신이 같은 스킬로 함께 공격 |
| 메아리 | 스킬 사용 시 일정 확률로 같은 스킬 1회 자동 재시전 |
| 자석술사 | 드랍 자동 흡수 + 흡수 범위 ↑, 범위 내 처치 시 추가 스탯 구슬 |
| 마노 | 달팽이 던지기 타수 12 |
| 마나실드 | n초마다 방어막 1회 생성 |
| 큐브마스터 | 스킬 선택 시 제시 옵션 1회 재추첨 가능 |
| 연성술사 | 스탯 구슬 획득 시 25% 확률로 동일 구슬 1개 복제 |
| 처형자 | 체력 15% 이하 적 처치 확률 ↑, 처치 시 3초간 공격력 +15% |
| 가속 | 적 처치마다 공격 속도 +2% 누적 (최대 +40%, 피격 시 초기화) |
| 흡혈귀 | 가하는 모든 피해의 5%를 체력으로 회복 |
| 저격수 | 크리티컬 데미지 +50% |
| 재생력 | 3초간 피격당하지 않으면 초당 체력 회복 |
| 수호자 | 12초마다 받는 치명타 1회를 무효화 |
| 천부적 재능 | 이능 선택 시 선택지 1개 추가 (3종 → 4종) |

#### Hidden — bad (출현 확률 5%)

"저주처럼 보이지만 숨은 보상이 큰" 고위험·고보상 풀. 강한 제약 + 강한 대가가 한 쌍.

| 특성 | 효과 |
|------|------|
| 봉인된 자 | 전직을 1차까지만 가능 / 대신 시작 스탯 대폭 보너스 |
| 불운 | 드랍률 0 / 단 처치 경험치 2배 |
| 유리대포 | 최대 체력 −50% / 대신 모든 데미지 +50% |
| 외길 | 스킬 슬롯 2개로 제한 / 대신 보유 스킬 데미지 +60% |
| 금욕 | 회복 아이템 사용 불가 / 대신 적 처치 시 체력 소폭 회복 |
| 고독 | 링크 패시브를 받지 못함 / 대신 본인 전 스탯 +30% |
| 빈곤의 서약 | 영구 재화 획득 0 / 대신 스탯 구슬 획득량 2배 |
| 무모함 | 방어력 0 (피해 그대로 수용) / 대신 크리티컬 확률 +30% |
| 시한부 | 입장 후 제한 시간 초과 시 즉사 / 대신 경험치·드랍 대폭 증가 |
| 침묵의 맹세 | MP 소모 스킬 사용 불가 (평타·쿨다운 스킬만) / 대신 평타 데미지 3배 |
| 깨진 그릇 | 최대 MP 0 (MP 스킬 봉인) / 대신 스킬 쿨다운 −40% |
| 외눈 | 명중률 대폭 감소 / 대신 명중 시 데미지 2배 |
| 박애주의자 | 본인 데미지 −30% / 대신 링크 패시브 효율 3배 |
| 도굴꾼 | 보스에게 받는 피해 2배 / 대신 보스 처치 보상 3배 |
| 약속된 죽음 | 부활·회복 전부 불가 / 대신 전 스탯 +25% |
| 무게추 | 이동 속도 대폭 감소 / 대신 받는 피해 −40% + 넉백 면역 |
| 봉인구 | 4차 전직 불가 (3차까지) / 대신 스킬 데미지 +40% |
| 운명론자 | 이능 재추첨·특성 변경 전부 불가 (첫 결과 확정) / 대신 시작 스탯 +40% |
| 고행자 | 레벨업 시 스탯 자동 증가 없음 / 대신 시작 스탯 대폭 보너스 + 경험치 1.5배 |
| 순교자 | 체력이 낮을수록 데미지 급증(버서크 강화판) / 대신 자연 회복 불가 + 체력 상한 −30% |

#### average — 무난 (출현 확률 50%)

소폭 플러스 위주의 안전 풀. 빌드를 망치지 않는 무난한 보정.

| 특성 | 효과 |
|------|------|
| 균형 | 모든 스탯 소폭 증가 |
| 견습 | 경험치 획득 소폭 증가 |
| 절약가 | 스킬 MP 소모 감소 |
| 단단함 | 방어력 소폭 증가 |
| 강건함 | 최대 체력 소폭 증가 |
| 날렵함 | 이동 속도 소폭 증가 |
| 집중 | 명중률 소폭 증가 |
| 행운아 | 크리티컬 확률 소폭 증가 |
| 수련생 | 스킬 쿨다운 소폭 감소 |
| 회복가 | 회복 아이템 회복량 소폭 증가 |
| 근면 | 스탯 구슬 획득량 소폭 증가 |
| 민첩한 손 | 공격 속도 소폭 증가 |
| 끈기 | 체력 30% 이하일 때 방어력 소폭 증가 |
| 탐험가 | 드랍률 소폭 증가 |
| 명상가 | 최대 MP 소폭 증가 |
| 투지 | 공격력 소폭 증가 |
| 절제 | 받는 피해 소폭 감소 |
| 친화력 | 링크 패시브 효율 소폭 증가 |
| 신중함 | 첫 피격 1회 피해 무효 (런당 1회) |
| 적응력 | 적 처치 누적 시 전 스탯 아주 소폭 증가 (낮은 상한) |

#### bad — 불리 (출현 확률 18%)

소폭 마이너스 풀. 치명적이진 않지만 빌드에 부담을 주는 핸디캡.

| 특성 | 효과 |
|------|------|
| 저주받은 | 회복량 30% 감소 |
| 둔재 | 경험치 획득 감소 |
| 약골 | 최대 체력 감소 |
| 외골수 | 링크 패시브 효과 절반만 적용 |
| 굼뜸 | 이동 속도 감소 |
| 둔감 | 명중률 감소 |
| 유리몸 | 방어력 감소 |
| 낭비벽 | 스킬 MP 소모 증가 |
| 느림보 | 공격 속도 감소 |
| 박복 | 크리티컬 확률 감소 |
| 빈손 | 드랍률 감소 |
| 단명 | 자연·비전투 회복 불가 (회복 아이템은 가능) |
| 허약 | 공격력 소폭 감소 |
| 산만함 | 스킬 쿨다운 증가 |
| 옹졸함 | 스탯 구슬 획득량 감소 |
| 겁쟁이 | 보스전에서 받는 피해 증가 |
| 무딘 칼 | 크리티컬 데미지 감소 |
| 게으름 | 스탯 구슬 흡수 범위 감소 |
| 변덕 | 이능 선택지 3종 중 1종이 무작위로 비활성 (2종만 선택) |
| 저주의 표식 | 일정 주기로 짧게 디버프(소폭 능력 저하) 발생 |

#### Gamble — 도박 (출현 확률 12%)

뽑기의 긴장감을 강화하는 고분산 풀. 잘 터지면 OP, 안 터지면 핸디캡.

| 특성 | 효과 |
|------|------|
| 올인 | 공격력 대폭 ↑ / 방어력 대폭 ↓ |
| 양날검 | 크리 시 추가 데미지 큼 / 빗나가면 자해 데미지 |
| 럭키 세븐 | 7초마다 랜덤 버프 또는 디버프 발동 |
| 한 방 | 최대 체력 1/3로 시작 / 데미지 약 2배 |
| 도박꾼 | 적 처치 시 50% 확률로 드랍 2배 / 50% 확률로 0 |
| 룰렛 | 입장 시 전 스탯이 랜덤 재분배 (총량 ±30% 변동) |
| 복불복 상자 | 스탯 구슬이 큰 값 또는 0으로 드랍 |
| 회전룰렛 | 스킬 사용 시 10% 확률로 쿨다운 즉시 초기화 / 10% 확률로 쿨다운 2배 |
| 양극화 | 크리티컬 확률 대폭 증가 / 단 크리가 아니면 데미지 대폭 감소 |
| 폭탄 돌리기 | 처치 시 폭발(주변 적 피해) / 가끔 자신도 피해 |
| 운명의 동전 | 피격 시 50% 확률로 피해 무효 / 50% 확률로 피해 2배 |
| 광란의 질주 | 공격·이동 속도 대폭 증가 / 단 가끔 조작이 폭주(무작위 이동) |
| 한탕주의 | 영구 재화 평소 획득 0 / 클리어 시 일확천금(대량 지급) |
| 시한폭탄 | 시간이 갈수록 공격력 누적 증가 / 일정 시간마다 폭발 자해 피해 |
| 광부의 곡괭이 | 스탯 구슬 획득 시 낮은 확률로 잭팟(대량) / 보통은 소량 |
| 미친 과학자 | 이능 효과 2배 / 단 효과가 무작위로 결정(선택 불가) |
| 럭키/언럭키 데이 | 입장 시 그 런 전체에 행운(전 스탯+) 또는 불운(전 스탯−) 중 하나 부여 |
| 카지노 | MP 대신 '칩'으로 스킬 사용, 처치 시 칩 무작위 획득(많거나 적거나) |
| 블러프 | 체력 1로 시작 / 회피율 극대화(공격이 자주 빗나감) |
| 양날의 축복 | 좋은 특성 1개 + 나쁜 특성 1개를 동시에 무작위 부여 |

> 도박 특성은 잘 뽑으면 OP, 못 뽑으면 핸디캡 — 뽑기의 긴장감을 강화하는 장치.

### 2-4. 레벨

- 최대 레벨 21
- **1~15레벨**: EXP 파밍만으로 자유롭게 레벨업
- **16~21레벨**: 레벨별 해금 조건 충족 + EXP 급증
  - 해금 조건 예시: "16레벨 — 몬스터 게이트 3회 클리어", "18레벨 — 마왕 발록 처치"
  - 각 레벨의 구체적 해금 조건은 별도 설계 예정
  - 해금 조건은 **계정 귀속** — 한 번 달성하면 이후 모든 캐릭터에 적용
  - EXP 요구량은 여전히 캐릭터마다 파밍 필요 (해금이 레벨업을 보장하지 않음)
- 21레벨 콘텐츠: 미정

### 2-5. 전직 및 이능

전직은 총 4차. 각 전직 시 메이플 옛날 직업 스킬 3개 제시 → 1개 선택.

| 전직 | 조건 | 부가 효과 |
|------|------|---------|
| 1차 | 1레벨 (캐릭터 생성 시) | — |
| 2차 | 5레벨 + 전직 퀘스트 보스 클리어 | — |
| 3차 | 13레벨 + 전직 퀘스트 보스 클리어 | 랜덤 특성 1회 추가 지급 |
| 4차 | 18레벨 + 전직 퀘스트 보스 클리어 | — |

- 전직 퀘스트 보스 클리어: **캐릭터 귀속** — 캐릭터마다 직접 처치
- 보스 클리어 타임 기록: **계정 귀속** — 리더보드 반영
- 이능은 3레벨 단위(3·6·9·12·15·18·21)로 총 7회 제공

#### 이능 선택지

이능 발동 시 아래 3종 중 랜덤 제시 → 1개 선택:

| 이능 | 효과 |
|------|------|
| 스탯 재분배 | 보유 스탯 총량 유지한 채 4종에 재분배 |
| 추가 스탯 제공 | 특정 스탯에 고정량 추가 |
| 특성 변경 | 현재 특성을 새 랜덤 특성으로 교체 |

### 2-6. 링크 캐릭터 패시브

- 링크 캐릭터는 **15레벨 이상**일 때 본인에게 배정된 **링크스킬**(2-8 참조)을 공격대에 제공
- 15레벨 미만은 링크 효과 없음
- 링크 슬롯 2칸 → 서로 다른 링크스킬 최대 2개 중첩 (**동일 링크스킬은 중첩 불가**)
- 링크 마스터 특성 보유 시 링크 패시브 효율 +20% / 박애주의자 보유 시 3배 / 친화력 보유 시 소폭 증가
- 외골수 특성 보유 시 링크 패시브 효율 절반 / 고독 특성 보유 시 링크 효과를 받지 못함
- 링크로 쓰던 캐릭터도 언제든지 플레이어블로 전환 가능
- 미배치 슬롯은 부캐 육성 또는 링크 풀 비축 용도로 자유 활용

#### 링크 캐릭터 육성 흐름

```
새 캐릭터 생성 (링크스킬 1개 랜덤 배정)
    ↓
플레이어블로 직접 플레이 → 15레벨 달성
    ↓
링크 슬롯에 배치 → 배정된 링크스킬 활성화
```

> 특성(정체성) · 패시브 스킬(본인 전투 기믹) · 링크스킬(아군 지원)은 서로 독립된 3개의 100종 풀이다. 특성이 좋아서 링크로 쓰는 게 아니라, **링크스킬 자체가 생성 시 별도로 랜덤 배정**된다.

## 2-7. 패시브 스킬 (2-8과 함께 별도 100종 풀)

패시브 스킬은 특성과 무관하게 **전직 시 획득**하는 상시 발동 스킬이다. 각 전직(1·2·3·4차)에서 액티브 스킬 3택 1과 **별개로** 패시브 스킬도 3택 1을 제공 → 4차 전직까지 최대 패시브 4개 보유. 큐브마스터 특성/재추첨권으로 재추첨 가능.

성격: **본인이 조작하며 터뜨리는 전투 기믹** — 온-크리, 온-킬, 반격, 도트 등 트리거형 위주.

## 2-8. 링크스킬 (2-7과 독립된 별도 100종 풀)

링크스킬은 패시브 스킬과 무관한 별도 풀이다. **캐릭터 생성 시 1개가 랜덤 배정**되며, 그 캐릭터를 15레벨 이상으로 키워 링크 슬롯에 넣으면 배정된 링크스킬이 공격대(플레이어블)에 활성화된다.

성격: **배경의 아군이 꾸준히 대주는 지원** — 오라·경제·안전망·상황 버프 등 상시형 위주. 동일 링크스킬은 링크 2슬롯에서 중첩되지 않는다.

두 풀의 전체 목록은 아래 [부록 A. 패시브 스킬 100종]과 [부록 B. 링크스킬 100종]에 수록.

## 3. 전투 시스템

### 3-1. 조작

- 이동: 방향키 / 가상 조이스틱
- 기본 공격: 자동 평타(근처 적 자동 타격) + 수동 보정
- 스킬: 쿨다운 기반 버튼 (전직으로 슬롯 확장, 최대 4개)

### 3-2. 데미지 산출 (개략)

```
최종 데미지 = (공격력 × 스킬 계수) × 크리티컬 배율 − 몬스터 방어력
 · 공격력    : 힘(물리) / 지능(스킬)에서 산출
 · 크리티컬  : 행운(LUK) 기반 확률·배율
 · 공격 속도 : 민첩(DEX) 기반 → 초당 타격 횟수에 영향
```

### 3-3. 생존 자원

- **HP**: 0이 되면 영구 사망 (부활자 특성 보유 시 1회 부활)
- **MP**: 스킬 사용 자원 (지능이 최대치, 절약가 특성이 소모 감소)
- **회복 아이템**: 드랍으로 획득 (먹보 특성이 회복량 증가 / 버서크·금욕은 적용 불가)

## 4. 스킬 시스템

각 전직마다 메이플스토리 5차 이전 직업 스킬 3개 제시 → 1개 선택.
스킬 풀은 차수별 분리, 풀 내에서는 전 직업 통합 → 하이브리드 빌드 가능.

| 시점 | 스킬 풀 | 슬롯 합계 |
|------|---------|---------|
| 1차 전직 (생성 시) | 전 직업 1차 스킬 | 1개 |
| 2차 전직 (5레벨) | 전 직업 2차 스킬 | 2개 |
| 3차 전직 (13레벨) | 전 직업 3차 스킬 | 3개 |
| 4차 전직 (18레벨) | 전 직업 4차 스킬 | 4개 |

> 큐브마스터 특성 보유 시 스킬 선택 제시 시 1회 재추첨 가능.
> 외길(Hidden-bad) 특성 보유 시 스킬 슬롯 최대 2개로 제한.
> 위 표는 **액티브 스킬**(조작 스킬) 기준. 같은 전직 시점에 **패시브 스킬**도 3택 1로 별도 지급된다 (2-7 참조).

## 5. 몬스터 토벌

### 5-1. 토벌 전장

- 몬스터 처치 시 스탯 구슬 드랍 (힘/민첩/지능/행운 랜덤)
- 좌우 포털에서 몬스터 웨이브 소환
- 종종 중간 보스 난입 (막대한 경험치, 일정 확률로 게이트 입장권)
- 전장 클리어 보상으로 낮은 확률로 몬스터 게이트 입장권 지급

#### 난이도 4단계

| 단계 | 몬스터 스탯 배율 | 보상 배율 |
|------|--------------|---------|
| 하급 | ×1.0 | ×1.0 |
| 중급 | ×1.5 | ×1.4 |
| 상급 | ×2.25 | ×2.0 |
| 최상급 | ×3.4 | ×2.8 |

### 5-2. 몬스터 게이트 (특수 던전)

- 입장권 소모로 입장
- 스탯 구슬 획득량 토벌 전장 대비 대폭 상승
- 몬스터가 좌·우 양쪽에서 조여옴 / 스탯 약 3배, 수 약 2배
- 종료 지점에 게이트 보스 등장
- 보상: 이능 재선택권, 고급 스탯 구슬

### 5-3. 보스 레이드 (메인 목표)

- 이 게임의 최종 목표: 더 강한 보스를 순차적으로 클리어
- 보스는 메이플스토리 5차 이전 기존 보스 활용 (에셋 큐레이션)
- 보스 비주얼·애니메이션은 원작 에셋 사용 / 패턴·AI는 mLua로 단순화 구현
- 보스 클리어 시 다음 보스 해금
- 전직 퀘스트 보스 및 레벨 해금 조건 보스도 기존 메이플 보스 풀에서 선정

## 6. 재화 · 메타 진행

### 6-1. 재화 및 아이템

| 자원 | 획득 | 용도 |
|------|------|------|
| 스탯 구슬 | 몬스터·보스 처치 | 인게임 즉시 성장 (해당 런 한정) |
| 게이트 입장권 | 전장·보스에서 낮은 확률 | 몬스터 게이트 입장 |

### 6-2. 사망 & 메타 진행 (로그라이트 확정)

- 플레이어블 캐릭터 사망 = 영구 삭제. 부활자 특성 보유 시 1회 한정 부활.
- 사망 시 ENDING 연출 → 캐릭터 여정 기록 (도달 레벨/난이도, 처치 수, 생존 시간, 최종 빌드)
- **캐릭터는 소멸하지만 계정은 유지** — 런에서 정산된 영구 재화는 계정에 귀속
- 링크 캐릭터는 전투에 참여하지 않으므로 사망 대상 아님

#### 계정 영구 강화 예시

- 캐릭터 생성 시 좋은 특성 가중치 소폭 상승
- 시작 스탯 보너스
- 게이트 입장권 보유 한도 증가

## 부록 C. 설계 메모 (패시브 · 링크)

- **레이어 역할 분리**: 패시브는 온-크리/온-킬/반격/도트처럼 **터뜨리는** 기믹, 링크는 오라/경제/안전망처럼 **대주는** 지원. 같은 크리 계열이라도 패시브 "폭심(크리 시 폭발)" vs 링크 "조준 지원(크리뎀 +15% 상시)"로 성격이 다르다.
- **획득 확정 필요(택1)**: ① 패시브 스킬을 전직 3택으로 줄지, ② 이능과 번갈아 줄지, ③ 별도 패시브 슬롯을 둘지 — 현재 안은 ①(전직 3택). 링크스킬은 생성 시 랜덤 배정안. 원하는 방식으로 조정 가능.
- **수치 검토 1순위(패시브)**: 흡수(흡혈)·치명타 강화·불굴(치명타 무효)·처치 연쇄는 베이스라인에서 셀 수 있음.
- **수치 검토 1순위(링크)**: 후원(재화 +20%)·조준 지원(크리뎀 +15%)·최후의 인연(부활)·유대의 절정은 우선 조정 대상.
- **중복/과적 방지**: 링크는 동일 링크스킬 미중첩(기존 규칙). 패시브도 완전 동일 효과 2개는 합연산 상한을 둘지 검토.
- **레어도 연계(선택)**: 특성처럼 패시브·링크에도 등급(common/rare/legendary)을 얹어 전직 3택·링크 배정 시 가중치를 줄 수 있음. 필요하면 100종 각각에 등급 태깅해 드림.
